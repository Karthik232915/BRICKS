# 🚀 GoBricks

Welcome to the **GoBricks** project! This is a modern web application built with the best tools to help you build faster and better.

---

## 📌 Project URL

🌐 [https://gobricks.website](https://gobricks.website)

---

## 🛠️ Tech Stack

This project is built with:

- ⚡ **Vite** — Fast and lightweight build tool
- 🔷 **TypeScript** — Strongly-typed JavaScript
- ⚛️ **React** — Declarative component-based UI
- 🧩 **shadcn/ui** — Beautiful and reusable UI components
- 🎨 **Tailwind CSS** — Utility-first CSS framework
