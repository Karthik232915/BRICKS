**Industry:** Digital Services & Solutions

**Domain:** [www.gobricks.website](https://www.gobricks.website)

## 🚀 Features

- **Modern UI/UX**: Built with shadcn/ui components and Tailwind CSS
- **Responsive Design**: Optimized for all devices and screen sizes
- **Performance Optimized**: Integrated with Vercel Speed Insights
- **TypeScript**: Full type safety and better development experience
- **React Router**: Client-side routing for smooth navigation
- **React Query**: Efficient data fetching and state management
- **Supabase Integration**: Backend services and database
- **Contact Forms**: Interactive contact and inquiry forms
- **Portfolio Showcase**: Professional portfolio and work samples
- **Service Pages**: Detailed service offerings and solutions

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui components
- **Routing**: React Router DOM
- **State Management**: TanStack Query (React Query)
- **Backend**: Supabase
- **Deployment**: Vercel
- **Performance**: Vercel Speed Insights

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd BRICKS
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   bun install
   ```

3. **Set up environment variables**
   Create a `.env` file in the root directory:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the development server**
   ```bash
   npm run dev
   # or
   bun dev
   ```

5. **Open your browser**
   Navigate to `http://localhost:5173`

## 🚀 Deployment
